# Sistema de Administración de Alumnos

Este proyecto es una aplicación de administración de alumnos de una escuela. Proporciona una interfaz gráfica de usuario (GUI) para capturar, buscar, actualizar y eliminar datos de los alumnos. La información se muestra en una tabla y se puede realizar búsquedas por apellido, matrícula o DNI.

## Características

- Alta de alumno: Permite agregar un nuevo alumno al sistema ingresando los datos requeridos.
- Baja de alumno: Permite eliminar un alumno existente del sistema.
- Buscar alumno: Permite buscar un alumno por apellido, matrícula o DNI y mostrar los resultados en una tabla.
- Actualizar datos del alumno: Permite modificar los datos de un alumno existente.

## Requisitos de instalación

- Java Development Kit (JDK) 8 o superior.
- IDE de desarrollo Java.

## Cómo ejecutar la aplicación

1. Clona o descarga el repositorio en tu máquina local.
2. Abre el proyecto en tu IDE de desarrollo Java.
3. Configura el proyecto para utilizar el JDK adecuado.
4. Compila y ejecuta la clase principal del proyecto.
5. O simplemente abra el archivor .jar llamado "Planilla de Alumnos".

## Tecnologías utilizadas

- Java: Lenguaje de programación principal.
- Swing: Biblioteca gráfica para la creación de la interfaz de usuario.
- Git: Control de versiones para el proyecto.
